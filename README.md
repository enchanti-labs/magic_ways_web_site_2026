# Magic Ways Web Site 2026

## Project info

Magic Ways connects travelers with the magic of Mexico. Discover Magic Towns, connect with local businesses and live authentic experiences.

## Getting Started

Follow these steps to run the project locally:

```sh
# Step 1: Install the necessary dependencies.
npm i

# Step 2: Start the development server.
npm run dev
```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
